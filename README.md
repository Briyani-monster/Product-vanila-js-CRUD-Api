# Product CRUD Api

## links of api are

-   Gets All Products
    -   GET /api/products



-   Gets one Product
    -   GET /api/products/:id


-   create a Product
    -   POST /api/products 
    -   body have data for creation


-   update a Product
    -   PUT /api/products/id
    -   body have update data


-   delete a Product
    -   DELETE /api/products/id
